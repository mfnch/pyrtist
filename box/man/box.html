Box

A language designed to draw figures ...

...and to do whatever else!


The history behind the language...

Box was originally created to describe figures. Its ancient name was gdl (graphic description language), but I decided to change that name when I discovered that gdl is the name of another language, which has nothing to do with my own language. Box can be considered to be the second (really improved!) version of gdl.
